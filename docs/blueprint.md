# **App Name**: Blockify

## Core Features:

- Image Upload: Allow users to upload an image from their local device.
- Block Pixel Configuration: Allows user to configure the width of the output by specifiying number of horizontal blocks.  The tool will automatically compute the appropriate height to keep the image looking natural.
- Block Palette: Apply a conversion to the image so that only the concrete colors from minecraft are used in the image
- Image Preview: Provide a real-time preview of the converted image using HTML image display.
- Image Download: Enable users to download the converted image in PNG format.

## Style Guidelines:

- Primary color: Use a desaturated light blue (#ADD8E6) to reflect the clear skies often associated with Minecraft.
- Background color: Very light gray (#F0F0F0) to give a clean, unobtrusive backdrop that keeps the focus on the pixelated images.
- Accent color: Use a slightly brighter, analogous blue (#B0E2FF) for interactive elements to draw user attention without being too jarring.
- Font: 'Inter', a grotesque-style sans-serif, should be used for both headlines and body text. Its neutral, modern appearance is a good match to the engineered style of Minecraft.
- Use simple, blocky icons to maintain a consistent Minecraft-inspired aesthetic.
- Implement a clean, single-page layout for easy navigation and quick image conversion.
- Use subtle animations, such as a smooth transition during image conversion or a gentle scaling effect on hover for interactive elements.
